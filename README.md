# SIGI - Sistema Inteligente de Gestión de Inventario (Frontend)

Frontend de **SIGI**, una aplicación moderna para la gestión inteligente de inventarios.  
Construido con **React**, **Vite**, **TypeScript**, **TailwindCSS**, **shadcn/ui**, **Zustand**, **React Router**, **WebSocket** y **Axios**.

---

## 🚀 Tecnologías principales

- **React + Vite**: base del frontend, rápido y modular.
- **TypeScript**: tipado estático para mayor robustez.
- **TailwindCSS**: estilos utilitarios para diseño responsivo.
- **shadcn/ui**: componentes accesibles y personalizables.
- **Zustand**: gestión de estado ligera y escalable.
- **React Router**: navegación SPA.
- **WebSocket**: comunicación en tiempo real con el backend.
- **Axios**: cliente HTTP para consumir APIs.
